# MCP Server

Betterlytics exposes your analytics data through a [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server. This lets you ask AI assistants like Claude about your website's traffic in natural language. No dashboards or manual queries needed.

## Creating an API Token

1. Navigate to **Dashboard > Settings > MCP**
2. Enter a name for your token (e.g. "Claude Desktop")
3. Click **Create token**
4. Copy the token immediately, it is only shown once

## Setup

### Claude Desktop

Add the following to your Claude Desktop MCP configuration file:

```json
{
  "mcpServers": {
    "betterlytics": {
      "type": "streamable-http",
      "url": "https://betterlytics.io/api/mcp",
      "headers": {
        "Authorization": "Bearer btl_your_token_here"
      }
    }
  }
}
```

Replace `btl_your_token_here` with the token you copied.

### Claude Code

Run the following command in your terminal:

```bash
claude mcp add betterlytics \
  --transport streamable-http \
  --header "Authorization: Bearer btl_your_token_here" \
  https://betterlytics.io/api/mcp
```

Replace `btl_your_token_here` with the token you copied.

## What You Can Ask About

Once connected, you can ask your AI assistant about any of the following:

- **Traffic overview**: visitors, sessions, and pageviews over any time period
- **Page performance**: top pages, specific URL analysis, scroll depth
- **Devices and browsers**: desktop vs mobile vs tablet, browser breakdown
- **Geographic distribution**: visitor counts by country
- **Traffic sources**: where your visitors are coming from (search, social, direct, referrers)
- **UTM campaign tracking**: analyze traffic by source, medium, campaign, term, and content
- **Custom events**: track and analyze custom events you've set up
- **Trends over time**: daily or hourly breakdowns to spot patterns

## Example Questions

Here are some questions you can try:

- "How many visitors did I get this week?"
- "What are my top pages by pageviews over the last 28 days?"
- "Show me a daily visitor trend for the past 7 days"
- "Which countries do most of my visitors come from?"
- "What's the device breakdown: desktop vs mobile vs tablet?"
- "Which browsers do my mobile visitors use?"
- "How did my traffic change compared to last month?"
- "What are my top traffic sources this month?"

## Managing Tokens

View and manage your tokens from **Dashboard > Settings > MCP**. Each token shows its creation date and when it was last used. You can delete tokens at any time. Deleted tokens are immediately revoked.

## Security

- Tokens are hashed with SHA-256 before storage
- Each token is scoped to a single dashboard
- Tokens support optional expiration dates
- Requests are rate-limited to 60 per minute per site
