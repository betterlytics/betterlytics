# Overview

The official Betterlytics WordPress plugin brings privacy-first analytics to your WordPress site. Install it in minutes and start tracking visitors with no coding required.

## Features

- **Automatic Page View Tracking** - Every page visit is tracked automatically
- **Privacy-First** - No cookies, fully GDPR compliant
- **One-Click Setup** - Enter your Site ID and start tracking
- **Event Tracking** - Built-in support for 404 errors, site search, and file downloads
- **Performance Monitoring** - Optional Core Web Vitals tracking
- **Lightweight** - Minimal impact on your site's performance

<Callout type="info">
**Requirements:** WordPress 6.0 or higher and admin access to install plugins.
</Callout>
